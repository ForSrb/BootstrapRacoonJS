# BootstrapBugZ

### Setup
- Create new project based on this template.
- Open terminal and run init script.
``` 
$ ./init.sh
```

### Requirements
- [Gum](https://github.com/charmbracelet/gum)
- Java 17
- node lts (18.16.0)
- pnpm
- postgres
- redis
- docker
- git
- maven latest
- google-java-format 1.16.0.2
- postman
